# Asteroids (Java)

A Java-based Asteroids game built to explore game loops, object movement, collision detection, and basic physics systems.

## Features
- Player movement and shooting
- Asteroid spawning system
- Collision detection
- Score tracking

## Game Modes
- Classic Mode (default gameplay)
- Gravity Mode (in development) – adds physics-based movement influence

## What I learned
- Game loop architecture
- Object-oriented design in Java
- Collision detection systems
- Score/state management in games

## Status
Core game complete — adding experimental gravity mode
